# 英语口语卡壳时:先用简单句把想法说清,再逐步加细节

核心诊断:卡壳往往不是英语不够,而是**想法本身还没成形**——大脑在同时干两件事(组织想法 + 找英文词),必然堵车。对策是把两件事拆开:先用最简单的方式把主意理清,再说一个简化版,细节后补。"先垒一块砖,不要试图一次举起整座城堡。"

## 拆开的三步

1. **先问自己三个问题理清想法**:我的主论点是什么?我为什么这么想?能举个例子吗?
2. **说最简版本**,短句,一句一个意思。比如解释第二大脑:"I have many thoughts. I don't want to lose them. I want to connect them. So I built this system."
3. **用思考短语给大脑争取时间**:Let me think for a second. / What I mean is… / The main idea is… / I'm trying to say that…

## 日常练习计划(存档自陪练给的 plan)

- **短句优先**:不追求完美句子,先让意思清楚
- **句型**:I think ___ because ___. / The reason why I ___ is because ___. / Before, I used to ___, but now I ___. / After I started ___, I can ___.
- **每天就一个话题说 1-2 分钟**:工作 / 项目 / 观点 / 日常
- **背短语不背单词**:capture my thoughts / organize my ideas / connect ideas together / generate new insights
- **忘词不停顿**,换个说法解释:"I don't know the exact word, but it means…"
- 目标不是完美英语,是把想法表达清楚、自然

## 陪练留下的开放问题

想不出内容的时候,到底是**不知道自己怎么想**,还是**知道观点但不知道怎么组织**?——这决定练的是思考还是表达,当时没回答。
