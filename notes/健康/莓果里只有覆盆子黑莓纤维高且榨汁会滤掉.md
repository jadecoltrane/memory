# 内容

直觉上会觉得莓果(蓝莓、草莓、覆盆子)果肉软绵绵的,不像苹果、芹菜那样有明显的"丝状纤维",所以觉得它没什么纤维。

**但要分品种看,不能一概说"莓果纤维高":**

- **真正高纤维的是覆盆子(~7g/100g)和黑莓(~5g/100g)**,在水果里确实名列前茅。
- **蓝莓(~2.4g/100g)、草莓(~2g/100g)其实并不高**,和苹果、橙子(~2g/100g 上下)差不多,别被"莓果=纤维炸弹"的笼统说法误导。
- **为什么吃不太出来**:这些纤维大多是[[concepts/膳食纤维|不可溶性纤维]],藏在表皮那层极微小的颗粒、以及果肉里密密麻麻的微小种子里(比如覆盆子/黑莓、桑葚里的小颗粒)。
- **但榨成果汁就完全不同**:哪怕是 NFC 果汁,工厂压榨过滤时这些小种子和硬质表皮都会被当"果渣"滤掉。本来纤维较高的覆盆子/黑莓,一旦变成果汁纤维也基本被扒得不剩——"果汁没什么纤维"这个直觉反而是对的。
- **想保留纤维怎么办**:把渣一起喝掉,或者用破壁机把水果连皮带肉打碎(Blending),纤维就还在,只是被切成了微小碎末,进入肠胃后依然能吸水膨胀、包裹糖分,起到延缓吸收的作用。榨汁(Juicing)才是流失纤维的那一步,不是"水果本身没纤维"。
