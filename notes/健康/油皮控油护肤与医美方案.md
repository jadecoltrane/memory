# 背景

用户 25 岁、油皮、毛孔粗大、有黑眼圈,出油程度为"洗完脸 1-2 小时内满脸油光"(重度皮脂腺亢进级别),曾就医开中药但自觉无效。

# 成分知识速览

- **水杨酸**:脂溶性,可深入毛孔溶解油脂、疏通粉刺,适合局部用于 T 区
- **壬二酸(15% 浓度为常用黄金浓度)**:5α-还原酶抑制剂,兼具控油、疏通毛孔、抗炎退红三重功效,是果酸的有效替代
- **烟酰胺 + 锌(PCA锌)**:锌为公认皮脂腺抑制剂,与烟酰胺搭配控油力强,但浓度高时刺激性也大
- **玻色因(Pro-Xylane,修丽可 A.G.E. 系列核心成分)**:抗糖化、修护屏障,机制与 A 醇不同但同样能改善毛孔和紧致度,刺激性远低于 A 醇
- **7% 甘醇酸(果酸)**:水溶性,主要作用于表层去角质,对深层毛孔油脂"疏通力不足",油皮场景中常被壬二酸/水杨酸取代
- 猛药叠加原则:同一晚不要叠加太多高浓度活性成分(如烟酰胺锌+果酸+壬二酸不要同天用),避免屏障受损、报复性出油

# 用户实际执行的早晚护肤流程

手上的产品:香奈儿水、修丽可双萃精华(SCF)、The Ordinary 10%烟酰胺+1%锌、The Ordinary 7%果酸水(后被建议闲置)、修丽可 A.G.E. 面霜/眼霜、希川科颜 15% 壬二酸。

- **早间**:洗面奶 → 香奈儿水 → 修丽可双萃(SCF) → 清爽哑光乳液/防晒。白天不用壬二酸,避免和底妆搓泥
- **晚间采取"隔夜交替法"**,壬二酸和烟酰胺锌不同天用:
  - **A晚(单数天,控油抗糖常规夜)**:洗面奶 → 香奈儿水 → TO 烟酰胺锌(涂T区) → 修丽可 A.G.E. 面霜
  - **B晚(双数天,壬二酸专项治理夜)**:洗面奶 → 香奈儿水 → 希川科颜 15% 壬二酸 → 修丽可 A.G.E. 面霜。壬二酸取豆粒大小局部点涂在T区/毛孔粗大处,刺痒感5-10分钟内消失属正常;觉得刺激可先涂 A.G.E. 面霜打底、壬二酸涂最外层("乳液打底法")
- **The Ordinary 7% 果酸水建议彻底闲置**:壬二酸+双萃(含水杨酸)已覆盖去角质/疏通需求,7%果酸对当前肤质是"毁屏障"的累赘,可挪去擦身体(手肘/膝盖/鸡皮肤)

# 医美方向(声称对重度出油更有效)

- **肉毒素水光("黄金控油针")**:在水光基础上加微量肉毒素,阻断皮脂腺神经信号,声称做完 1-2 周内出油量明显下降,可维持约 3-4 个月
- **黄金微针/点阵激光**:射频热能可部分破坏皮脂腺、刺激胶原增生,用于毛孔改善
- 医美术后 1-7 天修复期,建议停用所有功效性精华(维C、烟酰胺、果酸、壬二酸),只用医用喷雾/敷料 + 温和面霜
- 重度不改善时的终极选项:口服异维A酸(泰尔丝),需皮肤科医生指导,有致畸性等风险,不能自行购买服用

# 预算分配思路(该对话给出的参考,非权威建议)

- 医美(约占65%)+ 护肤品(约占35%)的分配逻辑:医美是"治本"(控油),护肤品做"性价比"补充而非砸大牌
- 该在护肤品上多花钱的品类:核心功效精华(抗氧化/抗老)、高阶抗老面霜、眼部功能性产品——这几类的成分稳定性、促渗工艺、安全性差异大
- 可以省钱的品类:洁面、基础爽肤水、基础保湿乳、片状面膜——功能单一,平价替代效果差异不大
